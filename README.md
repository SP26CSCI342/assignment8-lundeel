## Local setup

1. Clone the repo
2. Copy `server/.env.example` to `server/.env` and fill in `MONGO_URI` + `JWT_SECRET`
3. From the root: `npm install` (client) and `cd server && npm install` (server)
4. Two terminals: `npm run dev` (root, client) + `npm run dev` (server)
5. Open http://localhost:5173

## What I learned during deployment

My two largest setbacks in this projects were the UI setup and the deployment. I already knew I was inexperienced with web development so I was eager to learn the quirks of React UI. This did prove to be one of the harder sections of the project for me. Although my largest problem was related to deployment. I was unable to complete the client server integration and I was not able to diagnose the problem. I think next time I would like to budget more time ahead of deployment so I can ask for help when I run into errors.
